# random-word

Random hiragana word genertor.

## Install and usage

1. `$ git clone https://github.com/trashtoy/random-word.git`
2. `$ cd random-word`
3. `$ composer install`
4. `$ php run.php`

Output sample:

```
けぬんつぃいぇみーくゎどぅもかでしずつぉまへだるぐべづぁづぉたんいぇぎーぐふぇねーろ ... (omit)
```
